# 동희산업 청구항 표시 프로그램

`app.py`는 아래 작업을 자동으로 수행합니다.

- `의견제출` PDF에서 거절이유 청구항과 청구항별 상세 사유를 추출
- `배터리케이스` PDF에서 청구항 1~18의 위치를 추출
- 청구항 제목 옆에 상태와 요약 사유를 표시한 새 PDF 생성
- 청구항별 상세 사유를 보는 HTML 보고서와 JSON 데이터 생성

## 실행 방법

```powershell
python app.py
```

## 생성 파일

- `output/동희산업_배터리케이스_청구항표시.pdf`
- `output/result.html`
- `output/claim_mapping.json`

## 참고

- 빨간 표시: 의견제출통지서에서 거절이유가 지적된 청구항
- 녹색 표시: 의견제출통지 시점 기준 특허 가능 청구항
